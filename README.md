# SQLite Admin CGI

SQLiteデータベースをWebブラウザから簡単に管理できるCGIスクリプトです。データベースの作成、テーブルの作成、データのCRUD操作、検索、CSVインポート、SQLクエリの実行などの機能を備えています。

## 特徴

- **シンプルなWebインターフェース**: Bootstrapを使用した直感的なUI
- **完全なCRUD操作**: データの作成、読み取り、更新、削除
- **検索機能**: 部分一致、完全一致、前方一致、後方一致
- **CSVインポート**: 既存テーブルへの追加または新規テーブル作成
- **SQLエディタ**: CodeMirrorを使用したSQLクエリ実行
- **セキュリティ**: ファイルパス検証とHTMLエスケープ

## 必要条件

- Python 3.7以上
- SQLite3 (Python標準ライブラリに含まれる)
- Webサーバー (Apache, Nginxなど) でCGIが有効化されていること

## インストール

1. このリポジトリをクローンまたはダウンロードします。

   ```bash
   git clone https://github.com/yourusername/sqlite_admin_cgi.git
   cd sqlite_admin_cgi
   ```

2. `admin.py` をCGI実行可能なディレクトリに配置します。

3. Webサーバーの設定で、`.py` ファイルをCGIとして実行するように設定します。

   Apacheの場合、`httpd.conf` または `.htaccess` に以下を追加:

   ```
   AddHandler cgi-script .py
   Options +ExecCGI
   ```

4. ブラウザから `admin.py` にアクセスします。

## 使用方法

1. ブラウザで `admin.py` のURLを開きます。

2. **データベースの作成**:
   - 「DB作成」フィールドに新しいデータベース名を入力し、「DB作成」ボタンをクリックします。

3. **テーブルの作成**:
   - データベースを選択後、「作成」タブでテーブル名とカラムを定義します。
   - 主キーの設定も可能です。

4. **データの操作**:
   - 「追加」タブ: 新しいレコードを追加
   - 「一覧」タブ: 既存のレコードを表示
   - 「編集」タブ: レコードの更新と削除
   - 「検索」タブ: キーワード検索

5. **CSVインポート**:
   - 「作成」タブの下部でCSVファイルをアップロードし、インポート設定を行います。

6. **SQL実行**:
   - 「SQL」タブで任意のSQLクエリを実行できます。

## セキュリティ注意事項

- このスクリプトはデモンストレーション目的で作成されています。本番環境での使用には十分なセキュリティ対策を追加してください。
- ファイルパス検証は実装されていますが、Webサーバーのセキュリティ設定を確認してください。
- データベースファイルはWebアクセス可能なディレクトリに配置しないことを推奨します。
